Responsive design is a way of developing web properties so that the device which is used for determining the way that how the site will be displayed. The first site to show the layout that fixed to browser viewport width was Audi.com launched in late 2001 which was created by a team of four members first is Jurgen Spangl , second is Jim Kalbach, third is Ken Olling and last the forth is Jan Hoffmann.(Referenced). First Responsive web design is very good as it makes easy for us to get flexible layouts, images, and media sheet queries for creating web page. There are number of responsive web design techniques that can be used to create website. Normally developers used major and minor width breakpoint to create the proper and responsive web page. Responsive web design is much preferred by developers as it makes text easier to read and images easier to view. Its layouts adjust automatically and fixed with any different device such as laptop, computer, phone and smartwatch. Increase in number of mobile users there is more demand of developers with knowledge of responsive web design.
